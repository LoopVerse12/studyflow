A simple full-stack task management application.

## Features
- Add tasks
- View tasks
- Persistent storage using SQLite

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite

## How It Works
Frontend communicates with Flask APIs which store and retrieve tasks from SQLite.

## How to Run
1. Install Flask
2. Run `python app.py`
3. Open frontend using Live Server
